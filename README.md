# Customized Table

This project is a customizable table built with React.

## Live Demo

You can view the live demo of the project at the following URL:
[https://react-customized-table.netlify.app/](https://react-customized-table.netlify.app/)

## Features

- Column Group  ✅
- Row Group     ✅
- Pagination
- Sorting
- Filtering
- Add row
- Delete row
- Edit row
- Add column
- Delete column
- Edit column
